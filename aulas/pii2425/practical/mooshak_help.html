Help about Mooshak

What is possible to on Mooshak?

  • Send a proposal of a solution (a submission) in the form of code, receiving an automated answer.
  • See problem statements.
  • ASk questions (on this course questions should be made on Discord and not on Mooshak)
  • See all made submissions, submitted code and detailed results of each submission.
  • See the rankings for each class volume of problems.
  • See help about the system.

How to submit code?

  • Create a file with the extension .c
  • Make the code to solve the problem 😁
  • Don't forget compile, execute and test the code on your computer
  • Using the interface, choose the file (Browse) and send it (Submit)
  • All data should be read from the standard input and should be written to the standard output
  • All lines in the input and output should be terminated with a newline character

What to the automated answers given by Mooshak mean?

At this course you will be able to see for each submission a detailed table containing the result for each test case on which your program was tested (you can access it by clicking on the result of the submission on listing). Some of the tests are public, meaning that if you fail them you will be able to see what was the input and the expected output).

Answer Meaning
Accepted The program produced the correct output for all test cases
Compile Time Error The system was not able to successfully compile the code
Runtime Error The program terminated prematurely, generating an error during the execution (e.g. segfault, division by zero, etc)
Time Limit Exceeded The program took more execution time than what was allowed
Memory Limit Exceeded The program used more memory than what was allowed
Wrong Answer The output of the program is not what was expected
Presentation Error The program seems to produce a correct output but with a wrong format (e.g. wrong spaces or newlines).
This message is only sent on obvious cases.
