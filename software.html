Software

Here are direct links for some of the software I created related to my research activity:

  gtrieScanner - Quick Discovery of Network Motifs

A software tool that uses the g-trie data structure to very efficiently count occurrences of subgraphs on larger graphs, determining if they are network motifs.
 

    Adapa - Automatic DAta PArallelism

A free reusable tool able to run parallel applications on multiple computing platforms (such as Condor). It is designed to work only with problems exhibiting data parallelism and is flexible enough to allow its users to use any familiar programming languages that they are acquainted with.
