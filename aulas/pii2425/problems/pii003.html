[PII003] Time Goes By

Time can seem endless or fleeting, depending on how you measure it.

The Problem

Write a program that, given the number of complete days D, calculates and displays the equivalent time in hours H, minutes M, and seconds S.

Input

The input consists of a single integer D representing the number of complete days.

Output

The output should consist of three lines:

  • The first line should display the total number of hours in the given days, formatted as: 
    D days: H hours
  • The second line should display the total number of minutes in the given days, formatted as: 
    D days: M minutes
  • The third line should display the total number of seconds in the given days, formatted as: 
    D days: S seconds

Constraints

The following limits are guaranteed in all the test cases that will be given to your program:

2 ≤ D ≤ 1000       The number of complete days

Example Input 1 Example Output 1 
2

2 days: 48 hours
2 days: 2880 minutes
2 days: 172800 seconds

Example Input 2 Example Output 2 
25

25 days: 600 hours
25 days: 36000 minutes
25 days: 2160000 seconds
