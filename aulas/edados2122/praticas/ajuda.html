Ajuda sobre o Mooshak

O que é possível fazer no Mooshak?

  • Enviar uma proposta de solução para um problema (submissão), recebendo uma resposta automatizada.
  • Ver o enunciado dos problemas
  • Fazer perguntas aos professores (no caso de Estruturas de Dados devem fazer as perguntas no Slack)
  • Ver todas as submissõs efectuadas, código submetido e resultados detalhados da submissão
  • Ver a classificação de todos os estudantes
  • Ver uma ajuda sobre o sistema

Como enviar uma proposta de solução?

  • Criar um ficheiro com a extensão .java com o mesmo nome da classe principal (a que contém o main a executar)
  • Fazer o código para resolver o problema :-)
  • Usando a interface, escolher o ficheiro com o código (Browse) e enviá-lo (Submit)
  • Deve sempre enviar o código ainda não compilado (o ficheiro .java e não o .class)
  • Todos os dados são sempre lidos do stdin e o output do programa é sempre para o stdout

O que significam as respostas automáticas dadas pelo Mooshak?

Nesta unidade curricular receber como feedback uma tabela contendo os resultados em cada um dos testes feitos ao seu programa, sendo que alguns testes são públicos (ou seja, se falharem ganham acesso ao caso de teste em si, sabendo qual o input e o output esperado).

Resposta Significado
Accepted O programa conseguiu acertar em todos os testes que foram feitos
Compile Time Error O sistema não conseguiu compilar o programa, tendo acontecido um erro ou um aviso
Runtime Error O programa acabou prematuramente, gerando uma excepção
Time Limit Exceeded O programa não terminou dentro do limite de tempo dado
Wrong Answer O output do programa não é igual ao esperado, pelo menos num dos testes feitos
Presentation Error O programa dá um output correcto, mas não correctamente formatado (por exemplo, com espaços a mais)
Esta mensagem só é enviado em casos óbvios
Invalid Submission O ficheiro enviado não tem uma extensão válida ou então foi feita uma submissão fora do interface.
Output Limit Exceeded O output gerado passou do limite de espaço necessário
Program Size Exceeded O código do programa é demasiado longo (normalmente será aceita um ficheiro até 100Kb)
Requires Reevaluation Por alguma razão não descrita atrás, o seu programa tem de ser re-avaliado
Invalid Function Foi usada uma função do Java não permitida ou outro erro não designado
