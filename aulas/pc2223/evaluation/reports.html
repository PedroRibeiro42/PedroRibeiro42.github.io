Goals and Content

The report should be delivered as a PDF file and can be in english or in portuguese.

It should contain a detailed explanation of three problems that you have solved during this course: two of them chosen by me (check at the bottom of this page the problem distribution) and another chosen by you (choose the one that you are most proud of having being able to solve, the one that gave you more work or the one that you think was the most complicated). For each problem, you should include in the report:

  • Brief explanation of the problem statement, showing what is asks for (what is the input and expected output).
  • Description of your algorithm (as if you were explaining to a colleague you you solve it) and of any auxiliary data structures that you used. If it makes sense, you are encouraged to add images and/or links to animations. You can also talk about the difficulties you have before being able to submit with success.
  • Indication as precise as possible of the temporal and spatial complexities of the algorithm you used and its various components.
  • Short analysis/description of the code you submitted (which should be included as an annex to the report) indicating, if applicable, of any more subtle technical implementation details; you should also clearly point out any sources and references that you used to complete your code submission.

Problem Distribution:

On your report you should always include these two problems plus another one that you can choose among those you have successfully submitted.
