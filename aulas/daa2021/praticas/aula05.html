Exercício 1) Programação Dinâmica - Contagens


Implemente e submeta uma solução para o problema [DAA 017] Pirâmides.

  • Comece por recordar os problemas "Pirâmide de Números" [slides 8 a 22] e "Obras na Estrada" [slides 46 a 50], dados nas teóricas. Este problema é no fundo um problema de contagem (como o das obras) numa pirâmide (como o dos números).
  • Seja count[i][j] uma matriz que diz quantos caminhos possíveis existem a partir da posição j da camada i. Como preencher esta matriz a partir de outros valores já calculados? Como percorrer a pirâmide para garantir que quando precisamos de um valor já o temos calculado?
  • No final onde fica o resultado que queremos escrever?
  • Cuidado com resultado M, porque pode não caber num int (M < 263). Use long em Java e long long em C/C++ (imprima com "%lld")

Exercício extra 1) Programação Dinâmica - Strings

Este exercício é um extra, e não é considerado essencial no contexto desta aula. Use-o se quiser consolidar os seus conhecimentos num problema um pouco mais complicado, pois implica ser eficiente (para além de correcto).


Implemente e submeta uma solução para o problema [DAA 019] Capicuas.

  • Este problema tem "semelhanças" com o da distância de edição
  • Seja minimo(i,j) a quantidade mínima de dígitos a adicionar para tornar capica o "subnúmero" entre o i-ésimo e j-ésimo dígitos do número inicial
  • Trivialmente, minimo(i,i)=0 (um único dígito é uma capicua por si própria)
  • E os outros casos? Consegue escrever o resultado de minimo(i,j) em função de subproblemas mais pequenos?
  • Vou deixar ficar para já só estas dicas para ver como os alunos reagem a um problema novo onde têm de pensar "por vocês" :)
